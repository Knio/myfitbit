# MyFitbit

Because *"Your data belongs to you!"*

...and fitbit's own data export sucks.


## Installation

Recommended:

```sh
pip3 install myfitbit
```

Manual:

```sh
git clone git@github.com:Knio/myfitbit
cd myfitbit
python3 setup.py install
```
